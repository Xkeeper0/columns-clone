Block:draw (left, top, x, y, animationTimer)
Draw a block somewhere on the screen

Parameters

  • left: Leftmost X position
  • top: Topmost Y position
  • x: X position within grid
  • y: Y position within grid
  • animationTimer: Position in this block's animation series
Block:init (positions, width, height, imageWidth, imageHeight, gridWidth, gridHeight, animTime)
Initialize a new block type

Parameters

  • positions: Table of positions for this block's animations
  • width: Width of this block graphic
  • height: Height of this block graphic
  • imageWidth: Width of the base image
  • imageHeight: Height of the base image
  • gridWidth: Width of the playfield grid
  • gridHeight: Height of the playfield grid
  • animTime: Time each frame should be displayed
