Playfield:checkForClearAt (l, x, y, xs, ys)
Check for a specific clear from a given position All values are from the bottom-left-most block

Parameters

  • l: layer to check
  • x: Bottom-left X position
  • y: Bottom-left Y position
  • xs: X step (0 or 1, generally) (maybe higher values if you're weird)
  • ys: Y step (0 or 1) (don't be weird)

Return values:

  1. false if no clear at this location+step
  2. chain length, table of block locations
